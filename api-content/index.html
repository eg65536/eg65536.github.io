而堂吉诃德干了这么多傻事却不但没有变成丑角和傻瓜，相反却变的越来越伟大，甚至在屠格涅夫看来充满了理想主义的光辉，其原因就在于骑士精神本身。骑士精神本来是为了匡正世界的黑暗和不道德，约束为可以所欲为的骑士而创造出来的。在现实中生活的人们却从没有让它消失，相反人们一遍又一遍的用骑士纪功诗，用中世纪的抒情短诗和民谣，用骑士文学去颂扬它，就是为了让这些无所顾忌的老爷能够被打动，能够起来效法。所以骑士精神之所以显得可笑，正因为世道黑暗，而如果一个人起来用骑士精神行侠仗义，结果却是成了丑角，那么真正可笑的到底是骑士精神，还是堕落的世界呢？ （参见:为什么堂吉诃德如此执着于骑士精神） 我们的文艺理论中有许多这样对映的意象，比如酒神与日神、工具理性与价值理性……但堂吉诃德与哈姆雷特是不同的。在今天，它们使我们深思这样一个问题：理想主义者在价值多元的现代社会有怎样的意义？ （下文引文均出自钱理群《丰富的痛苦——堂吉诃德与哈姆雷特的东移》） 二、堂吉诃德的理想价值 如果我们认为现实意味着驯顺与达观，理想意味着幼稚与激情，我们到底是会做出我们自己的抉择，还是做一个踟蹰不前的哈姆雷特？自从人类抛弃刀耕火种走向社会与国家以来，这一问题就一直存在。我们总是想要达到大同社会，总是想要在地上建成天国，却又受社会的规训与惩罚所节制，虽生而自由却又无处不受限制。因此，当我们看到有理想的个人以其特质突破社会控制，实践自己的理想抱负时，被其强烈吸引——英雄崇拜由此产生。 屠格涅夫还进一步发展了海涅在《论浪漫派》里所提出的堂吉诃德与桑丘的关系的命题，将海涅所着意强调的精神与物质的关系引申为有思想、有理想的个人与群众的关系问题，并据此对堂吉诃德与哈姆雷特作了全新的考察。他指出：“哈姆雷特之流的的确确对群众毫无用处，他们什么也不能给予群众，不能把群众引导到任何地方去，因为他们自己哪里也不去”,何况他们 “还是蔑视群众的”。而堂吉诃德这样的“被心灵牵引着的人”，尽管群众（如桑丘）不断地“嘲弄，甚而加以诅咒和追击”他们，他们却不怕这一切，“勇往直前，全神费注于只有他们才看到的日标上面，探寻着，跌倒了又爬起来,终于找着了……”他们正是凭着这样的精神力量吸引着群众，使群众经常“处在幸福而诚实的迷惑状态之中 ”，进而对他产生一种忠诚。“完全依赖地跟随”他们走。 三、是的，我们都爱堂吉诃德……然后呢？ 是的，我们都爱堂吉诃德。但我们对于这种理想气概追随，正如上一节引文所言，很容易滑向狂热与偏执。在现代社会，这种不宽容的内在导向很容易导致更大的灾难。 很多时候，问题在于我们的理想主义到底是会填补我们心底的价值空虚，还是会在对过往时代的粉饰之中把我们又一次拖入前现代的苦难与不宽容？如果我们认为我们应该积极干涉社会，同时认为有一套凌驾于其他价值之上的绝对真理，我们到底是会把天堂拉到人间，还是把“人间天堂”变成地狱的同义词？ 批评家杜勃安留波夫在《真正的白天何时到来》里就明确指出，屠格涅夫笔下的罗亭、拉夫列茨基“他们是把新思想灌输到某部分人中间去的传播者，是启蒙者，宣传鼓动家”。……小说中的一个人物列兹涅夫(人们一般认为, 列兹涅夫的态度几乎就是作者的态度)很快就发现，人们在接受罗亭的新思想的同时，又不得不忍受着他的支配。屠格涅夫在罗亭形象上所概括的“启蒙者的专横”是一个极为重要的发现。……这并不仅仅是个人的品质，实际上是表现了俄国这样的文化落后的国家的启蒙主义者的某些特征的：他们的启蒙，并不足真正立足于唤起对象的自觉与独立自主，而是**以领袖、导师的身份，居高临下地将自己的主观思想灌输于对象。灌输即带有某种强制性，是干预式而非启发式的。这是一种隐藏在堂吉诃德式的热情无私教诲背后的专横，不仅启蒙者自身不自觉，连被启蒙者也很难察觉；即使像小说中的列兹涅夫这样，已经敏感到这种试图征服自己的强制力量，但仍然表示自己足倾心相从的,并且不无苦恼地要人们注意这一点——“是我同意他这样做的——妙就妙在这里”，因惑也正是在这里：这是崇高的理想、道德追求、目标下手段的专制，人们往往为了日标的崇高而觉、半自觉地接受这种专制。**一方面是一种人格的魅力的吸引，另一方面又是专制所必然带来的灾难性后果，人们常常因为这两者的巨大反差而在对堂吉诃德的专制的评价上产生摇摆。在一本《屠格涅夫传》（莫洛亚著）里，作者还提出了一个“专制主义的浪漫主义者”、“独裁政治的堂吉诃德”的概念。 而对于理想主义者而言，问题也是存在的：我们渴求社会进步，渴求人类自由意志的最大实现；我们尊敬群众，把群众捧到至高无上的地位；但当我们的人民不认同我们的看法，当我们的人民不认同我们的理念，我们对于人民的尊重又何以自处呢？我们真的能帮他们做决定，真的能自我标榜为唯一的真理，真的能理直气壮地怒斥群众的思想觉悟不够么？ 人们大概还记得,屠格涅夫在1860年的著名演讲里曾断言,尽管群众（桑丘们）不断地嘲弄,加以诅咒、追击堂吉诃德，但最终仍然会完全依赖地跟随他们走;但是，现在，当马尔凯洛夫到农民中去作宣传鼓动时，农民们非但没有跟着他走,反而将他捆起来送交官府,带头人竟是一位被他看作是“俄国老百姓的化身”的农民。这样的结局使马尔凯洛夫本人发生疑问：“所有那些社会主义者和思想家的著作、文章同书本(在他看来每个字都是天经地义的），这一切全是废话吗？”这也反映了作家的某些困感：屠格涅夫早已在写给赫尔参的信中认定：“您们崇拜的人民是十分的保守主义者”…… 这一矛盾使得理想主义者在现代变得极其危险：堂吉诃德大战风车不过是笑料，红色高棉呢？当我们不仅认为我们有正当理由改造社会，并且认为人民群众总体上是不够启蒙的时候，我们就自然而然会将自己对于大同社会的畅想强加于社会之上。在保守、停滞的前现代社会，改造社会的尝试受限于动员能力、生产能力的限制无法成为现实，但在现代社会呢？而当人民从进步的主体变为了保守的客体，当理想主义者从启蒙的传道者变为了光荣的引路人，我们应该如何防止地上天国变为人间地狱？我的答案是人道主义，是对人民意愿的尊重——即使它与我们心中的大同社会背道而驰。 四、人道主义的堂吉诃德 但罗亭却受到了曾经是他的尖铙批判者列兹涅夫的高度评价，这一点似乎出乎读者意料。列兹涅夫在大学里与罗亭同是波科尔斯基 “斯坦訊维奇”小组的成员,大概也是个堂吉诃德。但在小说中出现时，他已经懒懒散散、冷冷冰冰，他对别人说：“你老是需要火；但是火是毫无用处的。它闪一阵光，冒一阵烟，就完了。”尽管他承认火会“给人温暖”，但他更看到、也更重视：火“也在把人焚烧”。但这位过去的堂吉诃德心中的火并没有真正熄灭;小说结尾处，他才那么热烈地为罗亭身上的堂吉诃德气辩护：“他有热情，而这正是——请相信我，我自己就是个冷淡的人——这正是我们时代的最可宝贵的品质。我们大家都变得不可容忍地理智、淡漠，而且懒情了；我们沉睡了，僵冷了，所以只要有谁能够唤醒我们，温暖我们，哪怕只是一瞬间，我们也应该感谢他的”…… “也许你命该终身飘泊，也许，你正因此在完成着你自己还不知道的崇高的使命”，等等。列兹涅夫最后举杯高呼：“干杯！为青春，为青春的希望，憧憬，信念和真诚，干杯，为二十岁时我们的心曾为之跳动的一切，为比我们在生活中已经领略到和将要领略到的还更可贵的一切，干杯！ 诚如引文所言，火不仅给人温暖，也在把人燃烧。我们需要堂吉诃德，但我们需要的不是独裁政治的堂吉诃德，而是人道主义的堂吉诃德；是“拆下肋骨当火把”的堂吉诃德，而不是“最最亲爱的伟大领袖”的堂吉诃德；是坚持真理终身漂泊的堂吉诃德，而不是叱咤风云一刻也不放开最高权力的堂吉诃德；是传道者堂吉诃德，而不是独裁者堂吉诃德……总之，我们当然需要理想主义者，但我们不需要不宽容的理想主义者，而是需要多元价值的理想主义者。现代的理想主义者不是为了用自己的理想重塑社会，而是为了不让社会的异化改造我们。 我们仍然需要改造社会，但需要在所有人的共识之上改造社会——不是因为我们保守，而是因为我们知道现代社会的力量强大到了多么危险的程度。 唯有如此，我们才不会在把地上建成天国的过程中走向地狱，才能真的在这个时代重拾理想主义，填补我们的价值虚无，达到个人价值的最大实现。归根结底，什么是社会主义？什么是个人价值的最大实现？就是所有人对所有人的宽容与尊重，就是堂吉诃德那句不朽的名言：
