C++程序设计 参考资料整合

· 2021-11-30 · # 资料整合

写在前面

  • 下面将以「教材->实用网址」为顺序简要介绍我在C++学习过程中使用/浏览过的参考资料。
  • 我认为编程语言是一门应用技能，练习的重要性大于教材。坦率地说，对于有志于转码的学生来讲，区区一门语言不值得花多少时间。或者说，如果一门语言都很难入门，那还是赶紧找个厂子上班吧，码农这碗饭早就是红海了。
  • 主要介绍中文教材，没有什么英文教材及简评。
  • 请读者仔细甄别选取，以免被多余噪音干扰了自身学习节奏。
  • 文中部分链接可能需要特殊网络环境。列表内课本均可于Z-lib/Libergen搜到。
  • 衷心希望能对读者有所裨益。

教材

  1. 谭浩强
    • 没啥好说的，大家都在骂，大家都在用，什么叫路径依赖啊（摊手）。
  2. 易学C++
    • 老师推荐的，但我没看，所以不评价。
  3. 《C++程序设计》简明讲义 华东师大 潘建瑜
    • 代码示例中有些错误，但整体阅读体验很好。
    • 下载
  4. More Effective C++
    • 延伸阅读，一些技巧性的介绍。

实用网址

  1. 现代C++教程
    • 属延伸阅读，不具入门价值。
  2. replit
    • 在线编译网站，确实做到了【开箱即用】，我的体验很好，推荐。
  3. stackoverflow
    • 其实没啥道理在这提，凑个数
