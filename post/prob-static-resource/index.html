# 概率论与数理统计 参考资源整合

2021-11-30 4 min read # 资料整合

写在前面

  • 下面将以「各校自编教材->课本->杂项资源」为顺序简要介绍我在概统学习过程中使用/浏览过的参考资料。此外也会列出一些各大高校通行的教材。
  • 以下内容皆以经济系本科生的视角评述，不同院系的读者可能会得出截然相反的评价。此外，由于我不擅长概率论——大概也就六七十分的水准——因此对于教材的述评难免有不全面之处，请持批判态度阅读。
  • 主要介绍中文教材，英文教材及简评可参考台大数学系教材书评。
  • 整理内容较多，请读者仔细甄别选取，以免被多余噪音干扰了自身学习节奏。
  • 文中部分链接可能需要特殊网络环境。列表内课本均可于Z-lib/Libergen搜到。
  • 衷心希望能对读者有所裨益。

各校自编教材

  1. 概统 科大统计与金融系 2008
    • 内容相对浅一些，一些例子举的非常形象，非常适合初学者自学。但由于数理推导内容相对较少，个人认为有学术志愿的同学还是不能当作主教材读。
    • 下载：网址 笔记
  2. 概统 复旦管院 徐勤丰 2008
    • 教学内容较为紧凑，内容也偏难，但介绍了一些实用的方法与定理，适合数理基础较高的同学。
    • 我找到的版本有不少排版问题，阅读体验不佳。
    • 下载：网址
  3. 概率论 成大 颜国勇 2012
    1. 对岸的概率论教材，我只读了极限定理那一章，挺好。此外这位老前辈的微积分教材也可在网路上查到。
    2. 下载：网
  4. 数理统计讲义 华南理工 何志坚
    • 由于课程设计为一学期，讲的内容要深一些。此外，由于涉及到许多R语言的上机内容，相对而言更适合统计系学生。不过许多内容颇为实用，编排也较合理，我挺喜欢。
    • 在线阅览：网址 笔记

课本

  1. 概统 陈希孺
  2. 概统 峁诗松
  3. 概统 盛骤 谢世千 潘承毅
  4. 概率论 何书元
    • 我校概率论部分使用的教材。
  5. 数理统计 师义民
    • 我校数理统计部分使用的教材。
    • 优点在于既有定理推导也有不少实例，但在我的阅读体验里二者结合的并不好，显得比较割裂，一口气读下来会觉得很不协调。此外由于编写年代与设计对象的问题，对于本科生来说书里有一些不必要的内容。所以，对于有时间舔图想要把整本书读完的同学，我觉得有这个时间不如去学点R语言……
  6. 测度与概率 北师大严士健
    • 以测度为出发点的概率论教材，类似 Probability and Measure(Patrick Billingsley) 或者 Measure Theory and Probability Theory 。
    • 如果要衔接这本书作教材，我认为最好大一学数分的时候就用Principles of Mathematical Analysis之类带测度论的书，这样不至于太过茫然无措。
    • 坦率地说，你要是能以我校大二的知识水平读通这本书，那你不看也照样能4.0
  7. 伊藤清概率论 伊藤清
    • 很陈旧的书了，sigma-代数都用的是完全加法族这个称呼。不过和上一本对照着看能半懂不懂地看懂一点。其实看不下去，但好歹看了，就列在这里
