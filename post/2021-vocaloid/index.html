2021年有趣的v曲二创

· 2021-12-30 · # 2021年终总结

2021年的v家可以说是“周虽旧邦，其命维新”了。

首先我可能得向读者朋友解释一下什么叫v曲，这里就借用萌娘百科的词条吧。

Vocaloid在中国的发展和衰退并不是一个特别令人感兴趣的话题，而且这一话题的讨论常被引向对禾念偶像化运营策略的批判，或是对国人音乐创作能力的质疑。但通过对v曲二创的梳理和评点，我们还是可以从二次创作的角度看到这一群体的内在活力与趋势。

不过，我姑且算是历史爱好者，因此选取的主要是历史向投稿。所以其实标题是不全面的，严格来说应该叫“2021年有趣的历史向v曲二创”才是。

1. 像天子一样呐

  2. 从调音角度并不算成功的一首历史向v曲翻调，其填词也有掉书袋之嫌。不过有一段还是很令人满意的，也是我今年心目中的年度填词：

    人生死有定数 岂有邦国万载不亡乎
    帝子同黎庶 太仓一粟 茫渺任沉浮
    七尺终入冢骨 唯余德名千秋不腐
    欲长遗万古 绢帛立篇著

    所谓青史留名很多时候真是以有涯随无涯的蠢事啊……但又有谁不想当那个伊卡洛斯呢？

  3. 此外，原曲像神一样呐本身也是很有意思的一首曲子。匹老板在调音与填词上的功底还是很足的啊。

3. 亚非燃烧吧

  2. 国际主义在今天的中国是一个很令人感到疏离的词。我们当然都知道天安门城楼上那段“世界人民大团结万岁”的标语，但让一个如今沙文主义狂热盛行的好战势力输出自以为的“先进”与“解放”只能让人看到无尽的人道主义危机——尤其是当我们看到简中互联网盛行的“尼哥”笑话与“肖万美国队长论”时。这一令人感到沮丧的背景使得国际主义在今天显得尤为苦涩而令人怀念。
  3. 无论是中文还是日文填词我都很喜欢。不过，作为历史爱好者还是需要指出：无论是纳赛尔的泛伊斯兰主义还是苏加诺、尼赫鲁的自主路线，其历史局限性都是相当明显的。不结盟运动声势固然极大，但其结果却不尽如人意，这与其成员各国的民族主义倾向是脱不开关系的。

4. 向黎明奔去

  2. 严格意义上其实不算v曲二创。但yoasobi的作曲ayase确实是v家p主出身，也会上传相应的v家版本，所以我认为广义上也可算作v曲翻调。

  3. TNO背景，民族主义色彩很强的一首翻调。虽说填词其实非常蹩脚，效果一大半是靠原曲的高质量，但有一句我觉得还是很有价值的：

    千载历史过往不知多少王侯将相，唯有人民才是滔天的巨浪

    我深以为然。

  4. 此外，这也是今天很多所谓毛粉精神分裂的地方：明眼人都能看出个人崇拜和长期执政与所谓“人民万岁”口号之间的巨大差异，更不用说前三十年尤其是那十年的巨大苦难了。我们无法否认这句口号本身的巨大价值，但说出这句口号的人最后真的践行它了么？十年动乱的无尽苦难真的是所谓理想主义的孤身一击么？果然人还是只愿意相信自己愿意相信的东西啊……

7. 简短总结：

总的来说，v家确实在衰退。不同于拥有相对成熟音乐教育体系的日本，在中国音乐创作的土壤本身就是贫瘠的，这也使这个爱好难以拥有足量的本土新血，文化输入的痕迹很重。在b站力图挤入主流传媒圈的当下，中文Vocaloid的未来是很令人忧虑的。

但我们仍然有如此多有创见的二创，偏见与仇恨尚不能让我们放弃发声。在辞旧迎新的当下，还是让我们乐观点吧。
