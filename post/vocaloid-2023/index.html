今年的术力口十选

· 2023-12-24 ·

  1. 人狂热者（人マニア）/原口沙輔

    • 非常美丽的一首歌。很能反映我们术力口人的精神状态（确信）。原神，启动；

    • 原口的初投稿。他投的第二首ホントノ也很行；

  2. 萨姆沙（ザムザ）/てにをは

    • 算是葱今年最喜欢的一首。c社再不升级引擎葱真要落后了；

    • じん的NEO也不错，Pjsk的三周年纪念曲。Jin真的很擅长调这种面向未来的歌；

    • Ayase给魔法未来写的HERO一度让我感觉很失望，但听了几遍之后又觉得意外的耐听。鉴定为比暴走P强；

  3. 超预告篇（ウルトラトレーラー）/マサラダ

    • 说实话，マサラダ的三作我都很喜欢。选这首的原因是3：00开始很燃，充分展现了红钻头SV的强大性能；

    • 小小的我和谎言舞者也很厉害。尤其是小小的我，立意很打动我，PV做的也真的很好。

  4. 强风大背头（強風オールバック）/ゆこぴ

    • mikufans的第一首（非本家投稿）术力口神话曲，虽然神话完没多久稿件就因为站方想炒这个话题而锁隐了；

    • 虽然ゆこぴ后面的投稿同质化很严重，但单就这一作而言，实至名归；

  5. 新人類/まらしぃ×じん×堀江晶太（kemu）

  • 触手猴是高品位的。vocacolle春当之无愧的第一名；

  • Frog96的ビビビビ是PV非常美丽的一首歌。可惜我不推星界所以NG；

  1. はなれあうゼロ / Sohbana

    • vocacolle春最美丽的PV。Sohbana目前为止最耐听的葱曲；

    • Sohbana作曲担当的トラエノヒメ 也很不错；

  2. 拝啓/Early

    • 今年无色透名祭最喜欢的新曲。红钻头你是真的火了；

    • D.J.ァネイロ的アイサイト･ラブPV很有意思，可惜调的有些乱了个人感觉；

  3. 俊达派对之夜（ずんだパーリナイ）/なみぐる

    • 毛豆精除了俊汁俊慰Futa以外最可爱的二创（喂

    • 同作者的ずんだシェイキング也很行；

  4. 大女優さん/いよわ

    • 想了很久胃弱p应该选哪一首，我个人是最喜欢这一首啦，但再生不太行的样子。

    • 其实一度想选无色透名祭那一首，但感觉那首调的太乱了，桃色钥匙又感觉比较套路，最后选的这首；

  5. 散り散り/読谷あかね

    • 本来以为会是今年再生数最高的红钻头曲了，毒性很强；

    • 也是很适合音MAD的一首曲子；

  6. 一些没有提到的投稿

    • Deco的兔子洞感觉没有和老匹合作的才不是恶魔耐听 。纠结之后决定两个都不选；
    • 不选生日曲和宝可梦曲是因为我觉得好几首水准都差不多，但没有很抓耳的。印象最深的分别是Deco的BluePlanet还有稲葉曇的电气予报；
    • 作为百合男子挺想选春卷饭的Zero Talking的，不选的理由主要是曲子本身没那么行；
