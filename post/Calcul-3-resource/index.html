· 2021-11-30 · # 资料整合

写在前面

  • 下面将以「教材->PPT->补充内容」为顺序简要介绍我在数分III学习过程中使用/浏览过的参考资料。数学分析为各校通识课，也是STEM的基本功，教材更是浩如烟海，网上各有评说，请自行查阅。
  • 主要介绍中文教材，英文教材及简评可参考台大数学系教材书评。
  • 请读者仔细甄别选取，以免被多余噪音干扰了自身学习节奏。
  • 文中部分链接可能需要特殊网络环境。列表内课本均可于Z-lib/Libergen搜到。
  • 衷心希望能对读者有所裨益。
